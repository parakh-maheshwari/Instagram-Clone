# Instagram Clone

This is a full-stack Instagram Clone project built using FastAPI, MongoDB, React (Vite), and Tailwind CSS.

## Features
- User authentication (signup & login)
- Create posts with image URL and caption
- Like and comment on posts
- Follow and unfollow users
- View user profiles and home feed

## Tech Stack
- Backend: FastAPI, MongoDB Atlas
- Frontend: React (Vite), Tailwind CSS
- API Communication: Axios

## Project Structure
- instagram_clone_backend – Backend APIs
- instagram_clone_frontend – Frontend application

## How to Run
1. Start backend using `uvicorn app.main:app --reload`
2. Start frontend using `npm run dev`
