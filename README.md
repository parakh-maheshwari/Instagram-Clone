# Instagram Clone

A full-stack Instagram-style application built using FastAPI and MongoDB for the backend, and React (Vite) with Tailwind CSS for the frontend.

## Features
- User authentication (signup & login)
- Follow / unfollow users
- Create posts with image URL and caption
- Like and comment on posts
- Feed showing posts from followed users
- Profile page with user posts and counts

## Tech Stack
- Backend: FastAPI, MongoDB Atlas
- Frontend: React (Vite), Tailwind CSS
- API Testing: Postman

## Project Structure
- instagram_clone_backend/
- instagram_clone_frontend/

## How to Run
1. Start backend (FastAPI)
2. Start frontend using `npm run dev`
3. Test APIs using Postman

This project is built for learning full-stack development.
